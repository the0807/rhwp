`본질`, `본질적`, `정합 영역`, `본 영역`, `본질적으로` 같은 표현을 한 응답에서 반복적으로 사용하지 말 것. 메인테이너에게 기계적 인상을 주고 가독성을 해친다.

**Why:** 작업지시자가 2026-05-07에 직접 지적함 — "당신이 사용하는 어휘 중 '본질'과 같이 기계적 언어가 과도합니다." 같은 단어를 한 문서/응답에서 수십 번 반복하면 의미가 비어 있다는 인상을 준다.

**How to apply:**
- 같은 표현이 한 문단/한 응답에서 2번 이상 등장하면 다른 단어로 대체
- "본질 cherry-pick" → "코어 cherry-pick" / "src 한정 cherry-pick" / "선택적 cherry-pick"
- "본질적 정합" → "정합", "부합", "맞음", "통과"
- "본질적 영역" → "핵심", "범위", "대상"
- "본 영역" → "이 영역", "해당 부분", 또는 그냥 생략
- 보고서·검토서·코멘트·대화 모두에 적용
- 문장이 자연스러운 한국어 산문이 되도록 작성. 반복되는 추상어 대신 구체적 명사·동사 사용
